# Raw Dataset

Place the authorized source dataset here as:

```text
MIMIC-III_ppg_dataset.h5
```

Raw dataset files are excluded from Git because they total approximately 57 GB
locally and exceed GitHub's storage limits. Obtain the data through its
authorized source and follow all applicable access, privacy, and usage terms.

Never commit credentials, access tokens, protected health information, or the
raw dataset files to this repository.
